  • How does the website work?

    The website fetches the user's IP address and picks a default location on the map while also providing up to 6 hours of weather information based on the acquired location while also allowing for the user to choose a custom location.

  • Is this website mobile responsive?

    Yes, this website is responsive based on the size of the screen and has 3 screen profile options.

  • Is this website for commercial use?

    This website is only for pesonal use and was developed as beginner web project.

  • Does this website have a backend?

    The website does not currently have a backend as it is just a concept project, but plans to add a backend in the future are present.
